# Microsandbox Nim SDK

A minimal Nim SDK for the Microsandbox project.

## Installation

```bash
nimble install microsandbox
```

Or add to your `.nimble` file:

```nim
requires "microsandbox >= 0.0.1"
```

## Usage

```nim
import microsandbox

# Print a greeting
let message = greet("World")
```
