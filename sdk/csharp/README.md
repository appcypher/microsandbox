# Microsandbox C# SDK

A minimal C# SDK for the Microsandbox project.

## Installation

```bash
dotnet add package Microsandbox
```

Or via the NuGet Package Manager:

```bash
Install-Package Microsandbox
```

## Usage

```csharp
using Microsandbox;

// Print a greeting
HelloWorld.Greet("World");
```
