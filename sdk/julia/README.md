# Microsandbox.jl

A minimal Julia SDK for the Microsandbox project.

## Installation

```julia
] add Microsandbox
```

Or directly:

```julia
using Pkg
Pkg.add("Microsandbox")
```

## Usage

```julia
using Microsandbox

# Print a greeting
greet("World")
```
