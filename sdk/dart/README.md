# Microsandbox Dart SDK

A minimal Dart SDK for the Microsandbox project.

## Installation

```yaml
dependencies:
  microsandbox: ^0.0.1
```

## Usage

```dart
import 'package:microsandbox/microsandbox.dart';

void main() {
  // Print a greeting
  greet('World');
}
```
